# OpenUniversity

## For a start
  0. create your own Branch, to not get conflicts with other contributors, and work there, you        can make branches again, but from your branch only
  1. make:
     - makemigrations
     - migrate
     - createsuperuser


- python manage.py runserver
  2. go to:
      - http://127.0.0.1:8000/admin
